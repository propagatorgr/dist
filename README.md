# Κλασματική Απόσταξη Πετρελαίου – Διαδραστική Προσομοίωση

Απλή διαδραστική προσομοίωση της κλασματικής απόσταξης του πετρελαίου για χρήση στη διδασκαλία (Β΄ Λυκείου).

---

## Περιγραφή
Η προσομοίωση δείχνει τη λειτουργία της κλασματικής απόσταξης σε στήλη απόσταξης.  
Καθώς αυξάνεται η θερμοκρασία, τα επιμέρους κλάσματα του πετρελαίου υγροποιούνται σε διαφορετικά ύψη και συλλέγονται.

Περιλαμβάνει:
- κύρια κλάσματα πετρελαίου
- ενδεικτικά θερμοκρασιακά όρια
- αριθμό ατόμων άνθρακα (Cₙ)
- βασικές χρήσεις κάθε κλάσματος

---

## Πώς χρησιμοποιείται
1. Ανοίξτε το αρχείο **HTML** με διπλό κλικ σε οποιονδήποτε browser.
2. Μετακινήστε τον ρυθμιστή θερμοκρασίας.
3. Παρατηρήστε:
   - ποιο κλάσμα ενεργοποιείται
   - σε ποιο ύψος συλλέγεται
4. Το κουμπί **«Εκκίνηση / Μηδενισμός»** επαναφέρει την προσομοίωση στην αρχική κατάσταση.
5. Το εικονίδιο **ℹ️** εμφανίζει οδηγίες χρήσης.

---

## Τεχνικά χαρακτηριστικά
- Ένα μόνο αρχείο **HTML**
- Χωρίς εξωτερικές βιβλιοθήκες
- HTML / CSS / JavaScript
- Συμβατό με:
  - υπολογιστές
  - projector
  - διαδραστικές οθόνες LG (touch)
  - tablets

---

## Σημείωση
Τα θερμοκρασιακά όρια και τα εύρη C είναι **ενδεικτικά** και εξαρτώνται από τη σύσταση του πετρελαίου.

---

## Εκπαιδευτική χρήση
Κατάλληλο για:
- εισαγωγή του θέματος
- εννοιολογική κατανόηση
- συζήτηση στην τάξη
- επανάληψη

Η κλασματική απόσταξη παρουσιάζεται ως **φυσική διεργασία** και όχι ως χημική αντίδραση.

---

## Άδεια
Μπορεί να χρησιμοποιηθεί ελεύθερα για **εκπαιδευτικούς σκοπούς**.
